    The Backend (Private): We process raw 3D files using Python & NumPy to generate optimized brick structures.

    The Data Handoff: The backend exports a lightweight JSON definition.

    The Frontend (This Repo): The web client parses the JSON and reconstructs the model using Three.js geometry for user interaction.
