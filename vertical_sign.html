Building Tips for Vertical Signs

  • Start with the base: Build the foundation first for stability
  • Layer by layer: Assemble from bottom to top following the printed guide
  • Check alignment: Ensure each layer is properly aligned before adding the next
  • Use printed baseplate: The STL export provides a custom-fit base for your sign
  • Consider lighting: Add LED strips behind the sign for dramatic effect

The STL baseplate can be 3D printed and provides a stable foundation. Use glue or double-sided tape to secure the brick sign to the baseplate.
